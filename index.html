Essay

Сначала мы начинаем с создания пустого объекта с именем "goals". Затем мы выбираем различные элементы на веб-странице и сохраняем их в переменные: `goalTitleInput` - поле ввода для заголовка цели, `goalDescriptionInput` - поле ввода для описания цели, `goalDateInput` - поле ввода для даты цели, `goalDateForm` - форма для ввода цели, `addedGoals` - контейнер для отображения списка целей. Мы добавляем прослушиватель событий к форме `goalDateForm` на событие "submit". Когда форма отправляется, выполняется определенная функция. Внутри этой функции мы предотвращаем стандартное поведение события (обновление страницы). Мы создаем пустой объект с именем `Data`. Мы заполняем объект `Data` свойствами: `goalTitle`, `goalDescription` и `goalDate`. Значения для этих свойств берутся из соответствующих полей ввода. Мы добавляем объект `Data` в список целей. Мы вызываем функцию для обновления отображаемого списка целей. Внутри функции для обновления списка целей мы создаем пустую строку с именем `goalsHtml`. Затем мы перебираем каждый элемент в списке целей. В цикле мы присваиваем текущий элемент переменной `dataObj`. Мы создаем HTML-код и добавляем его в строку `goalsHtml`, используя значения из объекта `dataObj`. Мы продолжаем добавлять HTML-код для каждой цели в списке. Мы обновляем отображаемый список целей, устанавливая строку `goalsHtml` внутри элемента `addedGoals`. Вне функции для обновления списка целей определена функция для удаления цели. Внутри функции для удаления цели мы выбираем текущий элемент, используя атрибут `data-goal-title`. Мы фильтруем список целей, исключая элементы с заголовками, соответствующими заголовку текущего элемента. Мы снова вызываем функцию для обновления списка целей, чтобы отобразить обновленный список после удаления цели.
